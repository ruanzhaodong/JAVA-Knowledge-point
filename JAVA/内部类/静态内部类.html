静态内部类基本概念

静态内部类是定义在另一个类内部，使用 static 关键字修饰的类。它和外部类实例没有关联，能够独立存在。

主要特点

  • 不依赖外部类实例，可直接创建静态内部类的对象。
  • 只能访问外部类的静态成员，无法访问外部类的非静态成员。
  • 有助于把相关的类组织在一起，提升代码的可读性与可维护性。

示例代码（Java）

                    
        class OuterClass {
            static int outerStaticVar = 10;
            int outerNonStaticVar = 20;
        
            static class StaticInnerClass {
                void displayStaticVar() {
                    System.out.println("外部类的静态变量: " + outerStaticVar);
                }
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                OuterClass.StaticInnerClass inner = new OuterClass.StaticInnerClass();
                inner.displayStaticVar();
            }
        }
