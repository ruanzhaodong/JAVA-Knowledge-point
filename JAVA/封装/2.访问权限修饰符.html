阿东的知识库

  • 默认（包级）

    • 类、属性、方法，无访问权限修饰符时。同一个包中才能进行访问。

  • public访问权限修饰符

    • 使用public修饰的类、属性、方法。任何一个包中，都是可以进行访问的。

  • protected访问权限修饰符

    • 使用protected修饰的内部类、属性、方法。同一个包中都可以进行访问，不同的包中只有子类可以进行访问。

  • private访问权限修饰符

    • 使用private修饰的内部类、属性、方法。同一个包中的类内才能访问。外部若要访问，需在类内提供公共的访问方法，以供外部调用后进行访问。

  • 可视化图

    • 作用域 public protected 默认（包级） private
    类内 ✔️ ✔️ ✔️ ✔️
    同一个包中类外 ✔️ ✔️ ✔️ ❌
    不同包中的子类 ✔️ ✔️ ❌ ❌
    不同包中的非子类 ✔️ ❌ ❌ ❌
