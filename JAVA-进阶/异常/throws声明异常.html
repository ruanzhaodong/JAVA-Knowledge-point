Java异常处理 - 声明异常（throws关键字）

throws的用法

用于在方法签名后声明该方法可能抛出的异常类型，格式为：

public 返回值类型 方法名(参数列表) throws 异常类型1, 异常类型2 {
    // 方法体
}

throws关键字的作用

  1. 显式告知调用者该方法可能抛出的异常类型，强制调用者处理异常（通过try-catch捕获或继续用throws声明）
  2. 将异常处理逻辑与业务逻辑分离，提升代码可读性和维护性

代码示例

声明异常的方法：

public void readFile(String path) throws FileNotFoundException {
    File file = new File(path);
    FileInputStream fis = new FileInputStream(file); // 可能抛出FileNotFoundException
}

调用方处理异常：

try {
    readFile("data.txt");
} catch (FileNotFoundException e) {
    System.out.println("文件不存在：" + e.getMessage());
    e.printStackTrace();
}
